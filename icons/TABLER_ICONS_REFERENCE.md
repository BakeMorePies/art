# Tabler Icons Reference Guide

A comprehensive reference for all available Tabler Icons for use in GitHub README files and web projects.

## How to Use Tabler Icons

### Basic Usage in Markdown
```markdown
<img src="https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/outline/[icon-name].svg" width="16" height="16">
```

### Usage in HTML
```html
<img src="https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/outline/heart.svg" width="24" height="24" alt="Heart">
```

### Available Styles
- **Outline**: `icons/outline/[icon-name].svg` (4,964 icons)
- **Filled**: `icons/filled/[icon-name].svg` (999 icons)

## Search Tips

### Finding Icons by Category
1. **Food/Baking**: Search for `cake`, `cookie`, `chef`, `oven`, `pie`
2. **Technology**: Search for `code`, `terminal`, `server`, `database`
3. **UI Elements**: Search for `arrow`, `button`, `menu`, `settings`
4. **Social**: Search for `brand-`, `share`, `comment`, `message`
5. **Tools**: Search for `tool`, `wrench`, `hammer`, `gear`

### Icon Naming Convention
- Use kebab-case (lowercase with hyphens)
- Brand icons start with `brand-`
- Multiple variants use numbers (`menu`, `menu-2`, `menu-3`)
- States use suffixes (`-off`, `-on`, `-active`)

### Common Suffixes
- `-off` - Disabled/inactive state
- `-2`, `-3` - Alternative versions
- `-outline` - Outlined version
- `-filled` - Filled version
